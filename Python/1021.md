# 程式閱讀題:說明底下程式執行結果

# For 迴圈(loop)
```
for name in ['orange', 'apple']:
	print(name)

結果:orange apple
```

```
for index in range(0, 6):
	print(index)

結果:0.1.2.3.4.5
```

```
使用For 迴圈(loop)計算1+2+3+.....100

程式
sum=0

for x in range(1,101):
  sum +=x
print(sum)
```
