# For 迴圈(loop)

```
使用For 迴圈(loop)計算1+2+3+4.....+100
程式
sum=0

for x in range(1,101):
  sum +=x
print(sum)
```
```
使用For 迴圈(loop)計算1+3+5+7.....+99
程式
sum=0

for x in range(1,101,2):
  sum +=x
print(sum)
```
```
使用For 迴圈(loop)計算1*3*5*7.....*99
程式
total=1

for x in range(1,101,2):
  total *=x
  
print(total)
```
